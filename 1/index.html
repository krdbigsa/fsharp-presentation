Krótko o F#

wieloparadygmatowy język programowania zawierający w sobie głównie cechy języka funkcyjnego, ale umożliwiającym także pisanie kodu imperatywnego oraz obiektowego. Jest językiem silnie typowanym zaprojektowanym w celu pisania prostego, solidnego i wydajnego kodu do rozwiązywania złożonych problemów. Jest uruchamiany na platformie .NET.

Główne cechy F#

  • Domyślnie immutable
  • Domyślnie brak null
    • typom stricte F# nie można przypisać wartości null
    • najczęstszym wyjątkiem jest typ string
  • Zaawansowane wnioskowanie typów (type inference)
  • Wszystko jest wyrażeniem (expression)
    • każde wyrażenie zwraca jakąś wartość
    • brak void
  • Nacisk na programowanie deklaratywne
  • Formatowanie kodu jest ważne
    • significant white space
  • Kolejność plików jest ważna
    • zapobiega cyklicznym zależnościom
    • pomaga w zaznajomieniu się z kodem w projekcie
  • Brak domyśnych konwersji typów

Unikalne możliwości F#

  • Pattern matching + Active Patterns
  • Discriminated unions
  • Computation Expressions
    • Wyrażenia sekwencji
    • Async Workflows
    • Własne DSL (farmer, saturn)
  • Units of measure
  • Type providers

Pit of success

A well designed system makes it easy to do the right things and annoying (but not impossible) to do the wrong things." - Jeff Atwood
